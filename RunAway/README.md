# Insect Army

Unityで制作した２作目のゲームを公開しているリポジトリです。

# DEMO

#### ゲームジャンル

アクション

#### ルール

・障害物を避けてゴールを目指すゲーム
・障害物に当たると残機が一つ減る
・残機が全て無くなるとゲームオーバー

#### 操作方法

・移動：A,D
・ジャンプ：Space

#### プレイ映像

https://youtu.be/ruhEUfy6MVc

Unityでビルドして実行して下さい
